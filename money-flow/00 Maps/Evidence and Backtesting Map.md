Up: [[00_Money_Flow_Command_Center|Money Flow Command Center]]

This map defines which evidence is canonical, which evidence is true-forward research, and which dashboard views are display-only.

Canonical packs are generated by backend Strategy Validation machinery. Dashboard date filters and chart JSON do not regenerate canonical packs.

## True-Forward Replay

Examples:

- SOR-EV2 stop/exit and rejected-signal replay.
- SOR-EV3 avoid sideways / low-volatility drilldown.
- MF-ORIG-EV1.1 and MF-ORIG-EV2 original-system replay.

True-forward replay must move candle-by-candle without lookahead, use only information available at or before the decision candle, track position state, include fees/slippage, use explicit fill assumptions, and keep production-rule changes out of runtime unless a later approved phase says otherwise.

## Diagnostic Overlays

Examples:

- SOR-EV1 completed-trade stop overlays.
- Lookahead diagnostic proxies.
- Reporting-only attribution.

Diagnostic overlays are useful for hypothesis triage. They are not production candidates unless later replayed true-forward.

## Dashboard Visualization

Dashboard surfaces use committed summary JSON and ignored local chart/trade JSON for founder review.

- Historical Replay chart JSON is display-only.
- Evidence Lab overlays are display-only.
- Audit Review reads committed EV-AUDIT1 summary JSON.
- Dashboard date filters are display-only recalculations and are not canonical evidence-pack regeneration.
- Date filters are browser/display recalculations from loaded data.
- Date filters are not canonical evidence-pack regeneration.

## Methodology Labels

| Label | Definition |
| --- | --- |
| `canonical_evidence` | Backend evidence-pack output from DB-imported candle truth. |
| `true_forward_replay` | Chronological replay with no lookahead. |
| `completed_trade_overlay_estimate` | Overlay on already-completed trades; diagnostic only. |
| `lookahead_diagnostic_proxy` | Upper-bound diagnostic that can use future context; not a candidate. |
| `dashboard_display_only` | Browser visualization or display recalculation. |
