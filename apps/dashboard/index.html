    Strategy Logic

    Current Money Flow v1.2 rule flow, including the real 1D sleeve added in SV2.0.

    01

    Readiness Gates

    Enabled family, active and strategy-eligible instrument, fresh market data, aligned indicators, latest candle, enough history, valid instrument mapping.

    02

    Trend Stack

    Entry requires EMA5 above EMA10 above SMA20. Holding fails when the stack breaks or price falls below EMA10/SMA20.

    03

    Momentum Checks

    RSI must sit inside the sleeve band, MACD must be constructive when confirmation is required, and price cannot be too extended above EMA5. Entries below the RSI floor are not allowed.

    04

    Decision

    No position can open only when all entry checks pass. Existing positions can hold, reduce on RSI trim, or close on MA/trend/MACD failure. Market-structure diagnostics are not entry filters.

    Entry Path

    1. Bullish alignmentEMA5 > EMA10 > SMA20
    2. RSI constructiveInside the sleeve floor/ceiling and below overbought
    3. MACD constructiveMACD > signal and histogram >= 0 when required
    4. Entry qualityPullback between EMA10 and allowed EMA5 extension, or continuation above EMA5 within extension limit
    5. ActionOpen, otherwise no_trade with the first failed reason code

    This is constructive momentum / controlled pullback logic, not a buy-deep-oversold-weakness system.

    Position Path

    1. MA breakClose when EMA5 <= EMA10, EMA10 <= SMA20, or price < EMA10
    2. Trend invalidatedClose when price < SMA20 or EMA5 <= EMA10
    3. MACD rolloverClose when MACD < signal or histogram < 0
    4. RSI trimReduce when RSI reaches the sleeve trim threshold
    5. OtherwiseHold while no exit condition is active

    Sleeve Settings

    Sleeve History RSI band Overbought Trim Max EMA5 extension MACD

    1h 35 bars 50-68 74 80 2.0% required

    SV2.0 adds `sleeve_1d` as an initial baseline, not an optimized parameter set. Existing 15m / 1h / 4h settings remain unchanged.
